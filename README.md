COVID-19 Global Trends Dashboard (Tableau)

1. Problem Statement
Visualize COVID-19 data to understand global trends in infections, deaths, and population impact across countries and continents.

2. Tools Used
- Tableau (Data Visualization)
- SQL (Data Source)
- Excel (Data Preparation)

3. Process
- Used cleaned and analyzed data from SQL project  
- Imported dataset into Tableau  
- Created interactive dashboard with maps, bar charts, and trend lines  
- Visualized global COVID-19 trends over time  

4. Dashboard Features
- Global summary of total cases, deaths, and death percentage  
- Map showing percentage of population infected by country  
- Bar chart comparing total deaths across continents  
- Trend line showing infection growth over time  

5. Key Insights
- Europe recorded the highest number of COVID-19 deaths among continents  
- Infection rates increased significantly during major waves (2020–2021)  
- The United States showed one of the highest infection percentages globally
